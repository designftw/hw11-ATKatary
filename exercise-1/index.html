Paper-Prototype

We add the following 5 animations

  1. The like / unlike buttons spin when clicked. This is to give some indication that you have reacted
  2. There is a fade in of the entire page. This is to give a more relaxed feeling to entering the app.
  3. Messages are highlighted upon hove to improve visibility and indicate which one was selected
  4. Messages bounce in, this is mainly to appear like the messages are "coming in"
  5. Typewriter effect for channel name indicates where channel name can be inputed
